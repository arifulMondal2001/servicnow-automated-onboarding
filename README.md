Project Name: Automated IT Onboarding (ServiceNow)

Description:
This project automates the IT onboarding process using ServiceNow.

Smart Form: Created a Catalog Item that uses UI Policies to show or hide the "Laptop Type" question based on user needs.

Automation: Built a workflow in Flow Designer that automatically creates tasks for the Service Desk and Hardware teams.
